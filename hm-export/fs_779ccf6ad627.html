Abb. 30: Menüpunkt „Change Sets“

Die Ansicht ist aufgeteilt auf eine Listenansicht mit den Change Sets [Link] links und einer Liste mit den darin enthalten Artefakten/Changes rechts. Die Liste mit den Artefakten/Changes dient nur der Übersicht und kann nicht bearbeitet werden.

Wenn Sie ein Change Set aufrufen, öffnet sich der Artefaktpaket-Assistent für das Change Set [Link].

Schaltflächen

Symbol

Schaltfläche

Beschreibung

„Neu“

[Einfg]

Öffnet den Artefaktpaket-Assistenten zur Erstellung eines neuen Changes Sets.

„Kopieren“

[Strg] + [Umschalt] + [C]

Öffnet den Bearbeitungsdialog zur Neuanlage eines Changes, wobei die Einstellungen und die Artefakt-Auswahl des markierten Change Sets übernommen werden. Bevor das Change Set bearbeitet werden kann, müssen Sie sich zunächst am gewünschten Quellsystem anmelden. Dieses System wird dann fest im kopierten Change Sets gespeichert und kann nicht nachträglich geändert werden.

Die Bezeichnung wird, mit hinzugefügter laufender Nummer, aus dem kopierten Change Set übernommen. Falls die Bezeichnung bereits mit einer freistehenden Zahl endet, wird diese um eins hochgezählt.

ACHTUNG: Wenn Sie als Quellsystem in der Kopie eine ganz andere IKAROS-Umgebung wählen (also nicht bloß eines von eventuell mehreren „Systeme“-Stammdateneinträgen für dasselbe tatsächliche System), wird es zu Fehlern beim Erstellen von Paketen und beim Transport kommen.

„In anderes TRMS transportieren“

[Strg] + [Alt] + [i]

Öffnet zunächst einen Dialog zur Auswahl der Zielsysteme und startet anschließend die Übertragung des Change Sets in die Ziel-TRMS-Systeme (wo es als schreibgeschütztes Change Set angelegt wird; darin enthaltene Changes werden ebenfalls schreibgeschützt im Ziel-TRMS angelegt [Link]).

„Transportieren“

[Strg] + [i]

Öffnet zunächst einen Dialog zur Auswahl der Zielsysteme und beginnt dann mit dem Transport und anschließend der Installation des Change Sets in diese Systeme. Wenn mehrere Zielsysteme ausgewählt sind, erfolgt der Transport sequenziell nacheinander und für jedes Zielsystem wird ein eigenes Protokoll geschrieben.

Beachten Sie, dass die Oberfläche in dieser Browser-Registerkarte bis zum Abschluss gesperrt bleibt. Während dieser Zeit wird ein Wartedialog angezeigt. Erst bei Abschluss des Transports erhalten Sie eine Erfolgs- bzw. Fehlermeldung.

„Erstellen“

[Strg] + [P]

Startet die Erstellung der Paketdatei für das markierte Change Set und speichert sie zusammen mit dem Change Set in der Datenbank.

Das Paket wird binär in der Datenbank gespeichert.

Nach der Erstellung wird zudem der Download im Webbrowser gestartet, sodass Sie eine Kopie des Pakets lokal ablegen können.

„Herunterladen“

[Strg] + [Umschalt] + [E]

Startet den Download des „.trms4“-Pakets zum Change Set. Dies ist nur möglich, wenn zuvor ein Paket erstellt wurde.

„Filtern“

[Leertaste]

Ruft den Filterdialog für die Auswahlliste auf.

„Aufheben“

[Strg] + [R]

Hebt die aktuellen Filtereinstellungen auf.

„Sichten“

[F6]

Ruft den Verwaltungsdialog für benutzerdefinierte Sichten auf.
